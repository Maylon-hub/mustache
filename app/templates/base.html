About MustaCHE v2

MustaCHE (Multiple Cluster Hierarchies Explorer) is a tool for analyzing hierarchical density-based clustering results.

Version Information

Version: 2.0 (Rebuilt)

Technology Stack: Flask + Python 3.11 + scikit-learn + Plotly.js

Features

  • HDBSCAN density-based clustering
  • Interactive visualizations (Reachability, Dendrogram, 2D Map)
  • Ground truth validation metrics (ARI, AMI)
  • Multiple distance metrics support
